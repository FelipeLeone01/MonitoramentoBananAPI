# 🍌 Monitoramento de Temperatura - Banana Nanica

## 📌 Sobre o Projeto
Sistema IoT utilizando Arduino e sensor LM35 para monitoramento térmico em câmaras de armazenamento e maturação de bananas.

## 🎯 Objetivo
Garantir que a temperatura permaneça nas faixas ideais:
- Conservação: 13°C a 15°C
- Maturação: 15°C a 24°C

## 🧱 Estrutura do Projeto

- 📁 database → Scripts SQL
- 📁 frontend → Dashboard
- 📁 backend → Integração (futuro)
- 📁 docs → Documentação

## ⚙️ Tecnologias
- MySQL
- Arduino (LM35)
- HTML / CSS / JS
- Chart.js

## 🚨 Funcionalidades
- Coleta de temperatura em tempo real
- Armazenamento em banco de dados
- Sistema de alertas automático
- Dashboard com gráficos
